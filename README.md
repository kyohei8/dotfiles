# Setup New Mac

Macを買ったら最初にすること

* setting to System Preferences
 * setup trackPad
 * change language(ja->en)

* generate ssh-key(and add key to Github)


* install packages

```
$ cd ~
$ git clone [this rep]
$ sh dotfiles/install.sh
```

* Manual install
